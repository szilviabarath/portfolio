If you've searched for simple to use sphere tag-cloud - you have just find one!

Simply add one script file to your page, provide an array of tag name-id pairs, create "Clouder" object... And... Seems that's all you need!

No Cumulus, No jQuery, No Canvas, No any other horrible words :-)
